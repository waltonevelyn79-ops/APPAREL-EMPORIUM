# Admin User Guide — GlobalStitch Sourcing Website

## How to Login
1. Go to `https://yourdomain.com/admin/login`
2. Enter your email and password
3. You'll see the Dashboard

## How to Add a New Product
1. Go to **Products** → **Add New**
2. Fill in: Product Name, Category, Description
3. Upload product images (drag & drop or click to browse)
4. Fill in specifications: Fabric, GSM, Colors, Sizes, MOQ, Lead Time
5. Toggle "Featured" if you want it on homepage
6. Click "Save Product"

## How to Add Products in Bulk (Excel)
1. Go to **Products** → **Bulk Upload**
2. Download the sample Excel template
3. Fill in your product data in the template
4. Upload the filled Excel file
5. Preview the data and confirm import

## How to Change Website Content
### Change Company Information
**Settings** → **General** → Update company name, address, phone, email

### Change Homepage Slider
**Homepage Builder** → **Hero Slider** → Edit slides, change images, text, buttons

### Change Menu/Navigation
**Menu Builder** → Drag to reorder, add/remove menu items

### Change Colors & Logo
**Theme Manager** → Use color pickers, upload new logo

### Add Blog Post
**Blog** → **New Post** → Write content, add cover image, publish

## How to Check Contact Inquiries
Dashboard shows recent inquiries, or go to **Inquiries** page for full list

## How to Manage RFQ (Request for Quote)
**RFQ Management** → Drag cards between columns to update status visually

## How to Add Tracking Pixels (Developer Only)
**Tracking & Integrations** → Enter Pixel IDs, toggle ON

## How to Update the Website
Contact your developer, or if you have Developer access:
**Maintenance** → **Updates** → **Check for Updates** → **Update Now**
